# 零基础 C 语言入门到进阶

一套面向零基础学习者的中文 C 语言教程，以 C17 为主要语言标准，从计算机与编译基础讲起，逐步延伸到内存管理、工程组织和基础数据结构。仓库同时提供课程正文与配套源码，方便边学、边写、边验证。

> [在线阅读完整教程](https://superlxh02.github.io/c-study/)

## 课程特点

- **循序渐进**：从开发环境和基础语法开始，逐步进入指针、动态内存、多文件项目与数据结构。
- **文档与源码配套**：课程中的完整示例均可在 `src/` 目录找到对应代码。
- **重视内存安全**：围绕数组边界、指针有效性、对象生命周期和动态内存释放建立清晰的分析习惯。
- **兼顾工程实践**：除单文件程序外，还包含头文件组织、编译链接和 CMake 项目示例。

## 进阶篇（规划中）

入门篇已经覆盖核心语法、内存模型、基础数据结构和工程构建。后续计划继续补充以下内容：

- **CMake 进阶**：创建与导入库、编译参数配置、构建优化
- **函数接口设计**：使用 `void *` 设计通用接口，通过 `const` 和清晰的所有权约定减少误用
- **宏魔法**：利用预处理器消除样板代码、实现编译期分支和代码生成
- **内存对齐**：分析 `struct` 与 `union` 的成员布局，理解填充、对齐和空间优化方法
- **柔性数组**：在结构体末尾放置可变长度缓冲区，一次分配同时容纳元数据和数据
- **面向对象编程**：结合结构体、函数指针与虚表，探索封装和运行时多态的实现思路
- **树与图**：二叉树、平衡树、图的遍历与常见算法
- **实战项目**：待定

## 仓库结构

```text
.
├─ book/
│  ├─ book.toml             # mdBook 配置
│  └─ src/                  # 课程正文、目录与图片
└─ src/
   ├─ ch01-basics/          # 基础章节的独立示例
   ├─ ...
   ├─ ch16-headers/         # 多文件与头文件示例
   ├─ ch17-cmake/           # CMake 入门项目
   └─ ch18-data-structures/ # 六个独立的数据结构项目
```

课程正文位于 `book/src/`，配套代码位于仓库根目录下的 `src/`。数据结构项目采用 `include/`、`src/` 和 `CMakeLists.txt` 的常见布局，便于观察公开接口、内部实现与演示入口之间的关系。

## 本地构建与阅读

安装 [mdBook](https://rust-lang.github.io/mdBook/guide/installation.html) 后，在仓库根目录运行：

```bash
mdbook serve book --open
```

只生成静态站点时运行：

```bash
mdbook build book
```

生成的静态文件位于 `book/book/`。提交 `book/` 目录中的改动后，GitHub Actions 会自动重新构建并发布在线教程。

## 编译配套源码

包含 `main` 函数的单文件完整示例可以直接使用 Clang 编译：

```bash
clang -std=c17 -Wall -Wextra -Wpedantic path/to/example.c -o example
```

第十六章开始引入多文件项目，编译时需要同时指定相关源文件和头文件搜索路径：

```bash
clang -std=c17 src/ch16-headers/src/main.c src/ch16-headers/src/mathutils.c \
  -Isrc/ch16-headers/include -o mathutils_demo
```

第十七章及数据结构部分提供了 CMake 项目，可以使用统一流程构建：

```bash
cmake -S src/ch17-cmake -B build/ch17
cmake --build build/ch17
```

`src/ch18-data-structures/` 下的每个子目录都是独立的 CMake 项目；构建其他示例时，将 `-S` 后的路径替换为相应目录即可。
